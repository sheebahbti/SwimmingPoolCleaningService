# Swimming Pool Cleaning Service

A comprehensive service for managing swimming pool cleaning operations.

## Features

- Users and Cleaners Authentical and Authorization
- Schedule Service
- Notification when service is created, reminder before started, started and ended
- Email Service where Cleaners should be able to send the before and after pictures 
- Payment processing
- Dashboard for Admin review
- Deployment so that the url is public 
- Data Backups incase of failures
